# GitHomework21
这是开源文化的期末Git作业

这是一个用于GitHub实践练习的简单项目，包含基础的Git操作、分支管理及协作互动功能。

# 项目介绍

本项目是个人GitHub实践作业，主要目标：

- 熟悉GitHub仓库创建、文档管理流程
- 掌握Git分支操作（创建、合并、冲突解决）
- 体验GitHub协作功能（Issue、Star、Watch）
